# Copy-Paste These into Railway Variables

## Instructions:
1. Go to Railway Dashboard
2. Select your project "nish-leave-mgt-system-production"
3. Click "Variables" tab
4. Click "New Variable" for each one below
5. Copy and paste exactly as shown
6. Save

---

## Environment Variables (Update APP_URL with your actual domain):

```
APP_ENV=production
```

```
APP_DEBUG=false
```

```
APP_URL=https://nish-leave-mgt-system-production.up.railway.app
```

```
SESSION_DRIVER=database
```

```
SESSION_SECURE_COOKIE=true
```

```
SESSION_HTTP_ONLY=true
```

```
SESSION_SAME_SITE=lax
```

---

## After Adding Variables:

Railway will automatically redeploy. Wait for:
- Build to complete
- Deployment to succeed
